Base URL

{{ request.build_absolute_uri }}/api/

All API endpoints are relative to this base URL. Authentication is optional but recommended for higher rate limits.

Rate Limits

  • Anonymous users: 100 requests per hour
  • Authenticated users: 1000 requests per hour

Rate limit headers are included in all responses: X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset

POST Create Short URL
/api/urls/

Create a new shortened URL with optional custom alias.

Request Body:

{ "original_url": "https://example.com/very/long/url", "custom_alias": "my-link" // optional }

Response:

{ "id": 1, "original_url": "https://example.com/very/long/url", "short_code": "abc123", "custom_alias": "my-link", "short_url": "https://yourdomain.com/my-link", "click_count": 0, "title": "Example Page", "description": "An example page", "domain": "example.com", "is_active": true, "created_at": "2025-08-15T08:30:00Z", "updated_at": "2025-08-15T08:30:00Z" }
GET List URLs
/api/urls/list/

Get a paginated list of shortened URLs with optional filtering.

Query Parameters:

Parameter Type Required Description
page integer Optional Page number for pagination
domain string Optional Filter by domain
days integer Optional Filter URLs created in last N days
GET URL Details
/api/urls/{short_code}/

Get details for a specific shortened URL by short code or custom alias.

GET URL Analytics
/api/analytics/{short_code}/

Get detailed analytics for a specific URL including click statistics and charts data.

Response includes:

  • URL information and metadata
  • Click counts (total, today, this week, this month)
  • Recent click details
  • Analytics data for charts (daily clicks, device types, browsers, countries)
POST Bulk URL Shortening
/api/urls/bulk/

Shorten multiple URLs at once (maximum 50 URLs per request).

Request Body:

{ "urls": [ "https://example.com/page1", "https://example.com/page2", "https://example.com/page3" ] }

Response:

{ "results": [ { "original_url": "https://example.com/page1", "short_url": "https://yourdomain.com/abc123", "short_code": "abc123", "status": "success" }, { "original_url": "invalid-url", "short_url": "", "short_code": "", "status": "error", "error": "Invalid URL format" } ] }
GET Overall Statistics
/api/stats/

Get overall statistics including total URLs, clicks, top domains, and recent activity.

Error Responses

All errors follow a consistent format:

{ "error": "Error message", "details": { "field_name": ["Specific field error"] } }

HTTP Status Codes:

  • 200 OK: Request successful
  • 201 Created: Resource created successfully
  • 400 Bad Request: Invalid request data
  • 404 Not Found: Resource not found
  • 429 Too Many Requests: Rate limit exceeded
  • 500 Internal Server Error: Server error
